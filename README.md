# Запуск базы данных локально

Для запуска базы данных на локальной машине выполните следующие шаги:

1. **Установите Docker**  
   Убедитесь, что Docker установлен и запущен на вашей системе.

2. **Перейдите в директорию с репозиторием**  
   Откройте консоль (терминал) и перейдите в папку, где находится ваш проект

3. **Запустите контейнеры**  
   Выполните команду:
   ```bash
   docker-compose up -d --build
   ```
